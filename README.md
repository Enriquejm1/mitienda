E-commerce de Venta de Videojuegos:
Desarrollo de una plataforma web de comercio electrónico para la venta de videojuegos, con funcionalidades de autenticación de usuarios, catálogo de productos y panel de administración.
Principales funcionalidades:
Implementación de registro e inicio de sesión de usuarios mediante correo electrónico y contraseña.
,Desarrollo de catálogo de videojuegos con búsqueda por nombre y filtrado por plataforma.
,Gestión de carrito de compras e integración de pagos mediante PayPal.
,Implementación de promociones, descuentos y cupones aplicables a productos.
,Desarrollo de módulo de perfil de usuario para la actualización de datos personales.
,Gestión de direcciones de envío, incluyendo selección de dirección principal.
,Visualización del historial de pedidos y seguimiento de órdenes del cliente.
,Desarrollo de panel administrativo con operaciones CRUD para productos, clientes, cupones, descuentos y ventas.
,Exportación de información a archivos Excel desde los diferentes módulos administrativos.
,Implementación de gestión de categorías y control de roles (administrador y usuario).
Tecnologías utilizadas: HTML, CSS, JavaScript,,Angular, MongoDB, Bootstrap y PayPal ,etc.
